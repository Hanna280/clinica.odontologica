# Sistema de Agenda - Clínica Odontológica

Sistema web desenvolvido em Python utilizando Flask e SQLite.

## Funcionalidades

- Login
- Dashboard
- Cadastro de pacientes
- Edição de pacientes
- Exclusão de pacientes
- Agenda de consultas
- Logout

## Tecnologias Utilizadas

- Python
- Flask
- SQLite
- Bootstrap

## Como executar

1. Instalar dependências:
pip install flask flask_sqlalchemy flask_login

2. Executar:
python app.py
